This is an initialization workflow for Antigravity.

Your task is to create and maintain the AGENTS.md file, which serves as the authoritative reference for AI agents in this repository.
AGENTS.md should explain:

- What agents exist and why
- What each agent is responsible for
- How agents are expected to interact with the codebase and with each other

If AGENTS.md already exists, update it thoughtfully rather than overwriting it blindly.
Preserve useful context, improve clarity, and align the document with the current state of the project.
